# Build project

## Dependencies

- make
- cmake => 3.15
- c++20 compiler
- [Raylib dependencies](https://github.com/raysan5/raylib/wiki/raylib-dependencies)
- PThreads (POSIX threads)

## Package manager

- [cmake cpm](https://github.com/cpm-cmake/CPM.cmake)

## Packages used

- [raylib v5.5](https://github.com/raysan5/raylib)

## Build

```bash
# build release
make release

# build debug version
make debug
```
